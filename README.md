# Tic-tac-toe
This repository contains a website to play tic-tac-toe, in order to get some experience with VueJs.


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
